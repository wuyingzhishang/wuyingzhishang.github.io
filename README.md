# 暗影至尊工具箱

一个功能强大的在线工具集合，采用科技极客风格设计，提供二维码生成、油价查询、汇率转换、文本处理等实用功能。

## ✨ 功能特性

### 📱 二维码生成
- 支持网址、文本内容转换为二维码
- 多种尺寸选择（150px/200px/300px/400px）
- 一键下载生成的二维码图片

### ⛽ 油价查询
- 支持全国各地区的实时油价查询
- 提供92号、95号、98号汽油和0号柴油价格
- 热门地区快速选择

### 💱 汇率转换
- 实时加密货币价格查询（USDT、TRX）
- 国际金额快速转换（K/M/B单位）
- 多API数据源保障，确保数据可靠性
- 手动刷新与自动更新机制

### 📝 文本处理
- 灵活的文本格式转换
- 支持自定义源格式和目标格式
- 智能后缀添加功能
- 批量文本处理与一键复制

## 🛠️ 技术栈

- **前端**: 原生 HTML5、CSS3、JavaScript (ES6+)
- **设计**: 科技极客风格，深色主题，渐变发光效果
- **API来源**:
  - 二维码生成 API (api.lykep.com)
  - 油价查询 API (api.nxvav.cn)
  - 汇率 API (api.xinyew.cn)
  - CoinGecko / Frankfurter API (备用)

## 🚀 快速开始

### 在线访问

直接访问：**https://wuyingzhishang.github.io/**

### 本地运行

```bash
# 克隆仓库
git clone git@github.com:wuyingzhishang/wuyingzhishang.github.io.git

# 进入目录
cd wuyingzhishang.github.io

# 启动本地服务器
python -m http.server 8000

# 浏览器访问 http://localhost:8000
```

## 🎯 使用说明

| 功能 | 操作 |
|------|------|
| **油价查询** | 输入地区名称或选择热门地区，点击查询 |
| **汇率转换** | 输入金额，选择K/M/B单位，自动显示转换结果 |
| **文本处理** | 设置格式规则，输入文本，点击处理 |
| **二维码** | 输入内容，选择尺寸，点击生成 |
